## Comment l'installer ?
* Cloner le dépot : 
    ```
    git clone https://github.com/helloHackYnow/The-Pyewton-sandbox.git
    ```
* Installer les dépendances :
    ```
    cd chemin/du/dépot
    ```
    * Windows :
        ```
        py -m pip install -r requirements.txt
        ```
    * Linux :
        ```
        python3 -m pip install -r requirements.txt
        ```
